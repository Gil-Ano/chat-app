# NovChat 💬

> A real-time chat application built with the MERN stack and Socket.io, featuring rooms, direct messages, file sharing and emoji reactions.

## 🌐 Live Demo

- **Frontend:** https://novchat.vercel.app
- **Backend API:** https://novchat-backend.onrender.com

## ✨ Features

- 🔐 JWT Authentication
- 💬 Real-time messaging with Socket.io
- 🏠 Multiple chat rooms
- 📩 Direct messages between users
- ⌨️ Typing indicators
- 🟢 Online/offline status
- 📎 File and image sharing
- 😊 Emoji picker
- 🔍 Message search
- 🔔 Unread message badges
- ✓✓ Read receipts
- 🚫 Block/unblock users

## 🛠️ Tech Stack

- **Frontend:** React, Vite, Tailwind CSS, Socket.io-client
- **Backend:** Node.js, Express, Socket.io, MongoDB
- **Auth:** JWT
- **Deployment:** Vercel + Render

## ⚙️ Local Setup

### Backend

\`\`\`bash
cd backend
npm install

# Add .env with MONGO_URI, JWT_SECRET, PORT

node seeder.js
npm run dev
\`\`\`

### Frontend

\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`
